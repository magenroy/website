  • Sifted colimits is sort of like “filtered colimits + geometric realizations (colimits of simplicial objects)”
    • Geometric realizations is like quotienting by equivalence relations
      • The 1-categorical version of this is taking reflexive coequalizers.
      • This is analogous to the fact that where as the 1-truncated (i.e. 1-categorical) sheaf condition requires only an equalizer, the full sheaf condition requires a totalization (limit of a cosimplicial object).
      • Examples:
        • Taking the geometric realization of a simplicial set corresponds to viewing the simplicial set as a simplicial space and then taking the colimit. The 1-simplices are like homotopical relations between the 0-simplices, so instead of the colimit making the endpoints of a 1-simplex equal, it just attached an interval between them.
        • The composite functor from simplicial objects in commutative rings which first views them as simplicial objects in (which are levelwise discrete), and then takes geometric realization, is an equivalence after localizing the domain along weak equivalences of underlying simplicial sets.
    • Filtered colimits is things which are sort of like sequential colimits (think nested unions)
